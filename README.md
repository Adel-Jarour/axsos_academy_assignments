# Axsos Academy Assignments

This repository contains a collection of assignments and projects completed during the Axsos Academy bootcamp. It is organized into different stacks, covering various aspects of software development, from web fundamentals to advanced Python programming and backend development with Django.

## Repository Structure

The repository is divided into two main sections:

### 1. Python Stack (`/python_stack`)

Focuses on Python programming, Object-Oriented Programming (OOP), and backend development.

- **Django**: Backend web development using the Django framework.
- **Python Fundamentals**: Core concepts of Python language.
- **OOP (Object-Oriented Programming)**: Implementation of classes, inheritance, and data structures like Singly and Doubly Linked Lists.
- **Algorithms**: Data structures and algorithmic challenges implemented in Python.

### 2. Web Stack (`/web stack`)

Covers the essentials of front-end development and foundational computer science concepts.

- **HTML & CSS**: Building responsive and styled web pages.
- **JavaScript**: Client-side scripting and interactivity.
- **Algorithms**: Fundamental algorithm challenges in JavaScript.

## Getting Started

To explore a specific assignment:

1. Navigate to the corresponding directory (e.g., `python_stack/python/OOP/singly_linked_lists`).
2. Read the local `README.md` within that folder for specific instructions and task details.
