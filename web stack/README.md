# Web Stack Assignments

This section contains assignments focused on the core technologies of the web. The modules cover everything from static page layout with HTML and CSS to dynamic interactivity with JavaScript and algorithmic problem-solving.

## Contents

### 1. [HTML & CSS](./html-css)
Foundations of web design and layout:
- Semantic HTML5
- CSS Flexbox and Grid
- Responsive Design (Media Queries)
- Modern CSS techniques (Gradients, Animations)

### 2. [JavaScript](./js)
Client-side logic and interactivity:
- DOM Manipulation
- Event Handling
- ES6+ Syntax
- Asynchronous JavaScript (Fetch, Promises)

### 3. [Algorithms](./algorithms)
Fundamental computer science algorithms implemented in JavaScript, focusing on:
- Array manipulation
- Sorting and Searching
- Recursion
- Data Structure fundamentals

## How to View
- **HTML/CSS Projects**: Simply open the `index.html` file in any modern web browser.
- **JavaScript & Algorithm Files**: Can be run using Node.js in the terminal:
```bash
node <file_name>.js
```
Alternatively, they can be tested directly in the browser console.
