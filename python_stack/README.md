# Python Stack Assignments

This section of the repository contains all assignments related to the Python development stack at Axsos Academy. The curriculum covers foundational Python programming, advanced Object-Oriented Programming (OOP) concepts, and full-stack web development using the Django framework.

## Contents

### 1. [Python Fundamentals](./python/python_fundamentals)
Basic Python concepts including:
- Data types and variables
- Control flow (loops, conditionals)
- Functions and modules
- List and Dictionary manipulation

### 2. [Object-Oriented Programming (OOP)](./python/OOP)
Advanced programming concepts and data structures:
- Classes and Objects
- Inheritance and Polymorphism
- Data Structures:
    - Singly Linked Lists
    - Doubly Linked Lists
- Modularization and TDD

### 3. [Django](./diango)
Full-stack backend development:
- ORM and Models
- Views and Templates
- Authentication and Authorization
- RESTful APIs

### 4. [Algorithms](./python/algorithms)
Logic challenges and computer science fundamentals implemented in Python.

### 5. [Environments](./environments)
Setup and configuration for Python virtual environments.

## How to Run
Most assignments in this stack can be run directly via the terminal:
```bash
python <file_name>.py
```
For Django projects, use:
```bash
python manage.py runserver
```
