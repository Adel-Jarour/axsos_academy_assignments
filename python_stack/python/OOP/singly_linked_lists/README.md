# Python OOP Singly Linked Lists Project

## Description

This project provides a Python implementation of a Singly Linked List. It includes various methods to manipulate and traverse the list, such as adding nodes to the front or back, removing nodes from the front or back, removing specific values, inserting at specific indices, and printing the list values.

## Features

- **Add to Back:** Appends a new node to the end of the list.
- **Add to Front:** Prepends a new node to the beginning of the list.
- **Remove from Front:** Removes the first node from the list.
- **Remove from Back:** Removes the last node from the list.
- **Remove Value:** Removes the first node with a specified value.
- **Insert At Index:** Inserts a new node at a given position.
- **Print Values:** Displays the elements of the list in order.

## Prerequisites

- Python 3.x installed on your system.

## Setup and Running the Project

To download only this project (Singly Linked Lists) without cloning the entire repository:

### 1. Clone only this file

Open your terminal and run these two command:

```bash
git clone --no-checkout https://github.com/Adel-Jarour/axsos_academy_assignments.git
cd axsos_academy_assignments
```

### 2. Enable sparse checkout

```bash
git sparse-checkout init --cone
```

### 3. Specify the folder path

```bash
git sparse-checkout set python_stack/python/OOP/singly_linked_lists
```

### 4. Checkout files

```bash
git checkout
```

### 5. Create a Virtual Environment

Run the following command to create a virtual environment:

```bash
python -m venv venv
```

### 6. Activate the Virtual Environment

- **On Windows:**
  ```cmd
  venv\Scripts\activate
  ```
- **On macOS/Linux:**
  ```bash
  source venv/bin/activate
  ```

### 7. Run the Program

Once the virtual environment is activated, you can run the main simulation file:

```bash
python singly_linked_list.py
```

### 8. Deactivate Virtual Environment

When you are done testing the application, you can deactivate the virtual environment by simply running:

```bash
deactivate
```
