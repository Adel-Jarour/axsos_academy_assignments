# Python OOP Doubly Linked Lists Project

## Description

This project provides a Python implementation of a Doubly Linked List. It includes various methods to manipulate and traverse the list, such as adding nodes, deleting nodes, inserting at specific indices, finding the middle element, removing duplicates, and reversing the list.

## Features

- **Add to End:** Appends a new node to the end of the list.
- **Delete Node:** Removes a specified node from the list.
- **Insert At Index:** Inserts a new node at a given position.
- **Find Middle:** Retrieves the middle node of the list.
- **Remove Duplicates:** Removes nodes with duplicate values.
- **Reverse List:** Reverses the entire doubly linked list.
- **Print List:** Displays the elements of the list in order.

## Prerequisites

- Python 3.x installed on your system.

## Setup and Running the Project

To download only this project (Doubly Linked Lists) without cloning the entire repository:

### 1. Clone only this file

Open your terminal and run these two command:

```bash
git clone --no-checkout https://github.com/Adel-Jarour/axsos_academy_assignments.git
cd axsos_academy_assignments
```

### 2. Enable sparse checkout

```bash
git sparse-checkout init --cone
```

### 3. Specify the folder path

```bash
git sparse-checkout set python_stack/python/OOP/doubly_linked_lists
```

### 4. Checkout files

```bash
git checkout
```

### 5. Create a Virtual Environment

Run the following command to create a virtual environment:

```bash
python -m venv venv
```

### 6. Activate the Virtual Environment

- **On Windows:**
  ```cmd
  venv\Scripts\activate
  ```
- **On macOS/Linux:**
  ```bash
  source venv/bin/activate
  ```

### 7. Run the Program

Once the virtual environment is activated, you can run the main simulation file:

```bash
python doubly_linked_lists.py
```

### 8. Deactivate Virtual Environment

When you are done testing the application, you can deactivate the virtual environment by simply running:

```bash
deactivate
```
