# Sakila Database Assignments

This folder contains SQL practice assignments using the **Sakila** sample database.

## Contents
- **`sakila-data.sql`**: Full database dump including schema and data.
- **`sakila-quries.sql`**: SQL queries solving various practice problems.
- **`sakila-db-model.png`**: ER Diagram of the Sakila database.

## Queries Covered
The `sakila-quries.sql` file contains solutions for:
1. Finding all customers in city ID 312.
2. Listing all comedy films with their titles, descriptions, and ratings.
3. Fetching all films joined by actor ID 5.
4. Identifying customers in store 1 across specific cities (1, 42, 312, 459).
5. Retrieving 'G' rated films with 'Behind the Scenes' features for actor ID 15.
6. Finding all actors for film ID 369.
7. Listing drama films with a rental rate of 2.99.
8. Finding all Action films featuring actor Sandra Kilmer.
