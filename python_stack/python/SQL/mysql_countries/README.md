# MySQL Countries Assignments

This project involves querying a world database to extract demographic and geographic information about various countries and cities.

## Contents
- **`world.sql`**: Database dump containing `countries`, `cities`, and `languages` tables.
- **`world-quires.sql`**: SQL queries for data extraction and analysis.

## Analysis Tasks
The queries solve the following:
1. Identifying Slovene-speaking countries and their percentages.
2. Counting the number of cities in each country.
3. Listing major cities in Mexico (population > 500,000).
4. Finding languages spoken by more than 89% of a country's population.
5. Identifying densely populated small countries (Surface Area < 501, Pop > 100,000).
6. Filtering Constitutional Monarchies with specific capital and life expectancy criteria.
7. Detailed city reports for regions like Buenos Aires in Argentina.
8. Summarizing the number of countries per region.
